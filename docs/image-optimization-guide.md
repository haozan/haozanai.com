# 作品封面图片优化指南

## 📊 图片加载优先级

系统使用三层优先级来显示作品封面图片：

**① 外部 CDN URL（最快）** > **② 上传图片（优化后）** > **③ OG:Image 抓取**

---

## ⚡ 推荐方案：使用七牛云/CDN

### 为什么推荐使用外部 CDN？

- **加载速度最快** - 不经过 Rails 服务器，直接从 CDN 加载
- **节省服务器资源** - 减少服务器带宽和存储压力
- **全球加速** - CDN 节点分布全球，就近加速

### 如何使用？

1. 将图片上传到七牛云/阿里云 OSS/腾讯云 COS 等对象存储
2. 获取图片的 CDN URL（如 `https://cdn.example.com/cover.jpg`）
3. 在后台编辑作品时，粘贴到 **"External Cover URL"** 字段
4. 保存后，系统会优先使用这个 URL

---

## 📐 图片尺寸建议

### 推荐尺寸
- **分辨率**: 1200 x 628 px（比例 1.91:1）
- **文件大小**: < 200 KB
- **格式**: WebP（首选）或 JPG
- **质量**: 80-85%

### 为什么是这个尺寸？

- 1200x628 是社交媒体标准尺寸（og:image）
- 适配各种屏幕尺寸，包括 Retina 屏幕
- 保证清晰度的同时控制文件大小

---

## 🔧 三种图片来源对比

### 1. 外部 CDN URL（推荐）

**优点**:
- ⚡ 加载速度最快
- 🌐 全球 CDN 加速
- 💾 不占用服务器存储
- 🎯 完全可控（尺寸、质量、格式）

**缺点**:
- 需要单独管理图片
- 需要七牛云/OSS 账号

**适用场景**: 追求极致性能，有 CDN 资源

---

### 2. 后台上传图片

**优点**:
- 📤 操作简单，直接上传
- 🤖 自动优化（1200x628, 85% 质量）
- 🔒 图片存储在自己服务器

**缺点**:
- 🐌 相对较慢（走 Rails 服务器）
- 💾 占用服务器存储和带宽

**适用场景**: 图片数量少，追求方便

**优化建议**:
- 上传前手动压缩图片到 1200x628px
- 使用 WebP 格式可减少 30% 文件大小
- 推荐工具: TinyPNG, Squoosh, ImageOptim

---

### 3. OG:Image 自动抓取

**优点**:
- 🤖 全自动，无需上传
- 🔗 直接使用目标网站的图片

**缺点**:
- ⏱️ 首次抓取需要时间
- 🎲 图片质量取决于目标网站
- ⚠️ 依赖外部网站（可能失效）

**适用场景**: 快速添加作品，临时使用

---

## 📊 性能对比

| 方案 | 首屏加载时间 | 带宽占用 | 推荐度 |
|------|------------|---------|--------|
| 外部 CDN | ~100ms | 无 | ⭐⭐⭐⭐⭐ |
| 上传图片 | ~500ms | 高 | ⭐⭐⭐ |
| OG 抓取 | ~200ms | 无 | ⭐⭐⭐⭐ |

---

## 🎯 最佳实践

1. **新作品**：使用外部 CDN URL（七牛云/OSS）
2. **已有作品**：
   - 图片数量少（<10）：继续使用上传或 OG 抓取
   - 图片数量多（>10）：迁移到 CDN
3. **图片准备**：
   - 使用 Photoshop/Figma 导出 1200x628px WebP
   - 或使用在线工具压缩到 < 200KB

## 💡 常见问题

**Q: 我已经上传了图片，为什么还显示 OG:Image？**  
A: 检查是否上传成功。如果上传失败，系统会降级使用 OG:Image。

**Q: 可以同时使用三种方式吗？**  
A: 可以，但只有最高优先级的会生效（外部 URL > 上传 > OG 抓取）。

**Q: 如何删除已上传的图片？**  
A: 在编辑页面留空上传字段并保存，系统会保留原图片。若要删除，需在编辑时上传新图覆盖。

**Q: 七牛云收费吗？**  
A: 七牛云有免费额度（10GB 存储 + 10GB 流量/月），对于个人作品集完全够用。

## 🚀 快速开始

### 使用七牛云（推荐）

1. 注册七牛云账号：https://www.qiniu.com/
2. 创建对象存储空间（Bucket）
3. 上传图片（建议 1200x628px, < 200KB）
4. 复制图片的 CDN URL
5. 在后台粘贴到 "External Cover URL" 字段

### 继续使用上传

1. 准备图片（推荐 1200x628px）
2. 后台上传
3. 系统自动优化到 1200x628px, 85% 质量

### 使用 OG 抓取

1. 确保作品 URL 有效
2. 系统自动抓取 og:image
3. 可手动点击 "Refresh Cover" 重新抓取
